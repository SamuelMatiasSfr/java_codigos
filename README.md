# Códigos em Java

Repositório de códigos em Java! Estes projetos foram desenvolvidos como atividades práticas da matéria Linguagem e Técnicas de Programação 2 no curso técnico de informática do CEFET-MG.

## Tecnologias:
- Java
- Java Swing
- Java RMI
- LibGDX
- MySQL
- VS Code

## Como executar
1. Baixe e extraia o arquivo zip com os projetos
2. Abra o projeto desejado na IDE de sua preferência
3. Execute a classe do projeto que possui a função `main`
